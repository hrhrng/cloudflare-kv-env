# cfenv Docs

`cfenv` is a Cloudflare KV-based environment management tool for personal projects and multi-machine workflows.

It provides a `vercel pull`-style experience with both:

- A Node.js CLI package: `cfenv-kv-sync`
- A Python SDK package: `cfenv-kv-sync-python`

## What you can do

- Manage `.env` values in Cloudflare KV by project and environment.
- Pull env values to local machines quickly.
- Export env values for CI/CD and runtime injection.
- Use Node/Python SDKs for hot-update workflows.

## Published packages

- npm: `cfenv-kv-sync` (install with `@beta` for latest beta)
- PyPI: `cfenv-kv-sync-python`
